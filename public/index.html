311 Checkpoint 1

This backend express/mysql app includes the following routes:

GET/users

This route will provide a list of all the users from the database with an id, first and last name, address, city, county, state, zip, primary phone, secondary phone, and email address.

GET/users/:id

This route will provide you with a single user by id.

POST/users

This route will permit you to add an entry for a new user; user will be assigned an ID and returned to you as long as you provide the fist & last name, address, city, county, state, zip, primary phone, secondary phone, and email address.

POST/users/:id

This route will allow you to update a specific user's first and last names with the new information specified in the body of the request, provided they exist.

DELETE/users/first_name

This route will allow you to delete a user(s) from the database by specifiying the first name in the body of the request.
